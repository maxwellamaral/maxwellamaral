Processo SIMPLESTEC de Desenvolvimento de Software

1. Processo

O modelo do processo proposto para a Empresa é definido abaixo:

O Processo SIMPLESTEC de Desenvolvimento de Software é composto por quatro fases principais: Requisitos, Planejamento, Desenvolvimento e Liberação. Contém dois processos de entrada chamados Análise Comercial e Atendimento, que objetiva o levantamento inicial dos requisitos para um novo produto ou para a evolução do produto, respectivamente. Possui ainda três processos gerenciamento e acompanhamento chamados Gerência de Requisitos (GRE), Gerência de Projeto (GPR) e Gerência de Qualidade (GQA).

Processos modelados até o momento:
  • Atendimento
  • Fase: Requisitos (beta)
  • Fase: Planejamento (beta)
  • Fase: Desenvolvimento (beta)
