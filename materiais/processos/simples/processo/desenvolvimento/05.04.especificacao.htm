Detalhamento das atividades

05 - Desenvolvimento

05.04. Elaborar e/ou revisar especificação e interface visual
Finalidade
Esta atividade abrange outras atividades relativas à construção e detalhamento da especificação e da interface visual (prototipagem).
Observações
Esta atividade da fase de desenvolvimento contém duas sub-atividades:
  • Elaboração e/ou revisão da especificação do projeto;
  • Elaboração e/ou revisão da interface visual do projeto.
As atividades relacionadas à especificação ocorrem paralelamente ao projeto da interface visual — que será validado junto aos interessados do projeto.

05.04.02. Elaborar e/ou revisar Interface Visual
Artefatos de Entrada
VIS - Documento de Visão
REQ - Documento de Requisitos
DAQ - Documento de Arquitetura de Software
Artefatos de Saída
PRO - Protótipos da Interface Visual
Papel Responsável:
DSG - Designer
Papel Participante:
Nenhum
Ferramentas
Ferramenta de Desenvolvimento de Software
Navegadores WEB
Microsoft Visio
Microsoft Word
Adobe Photoshop
Adobe Flash
Passos
  1. Analisar requisitos e arquitetura do projeto;
  2. Identificar identidade visual;
  3. Desenvolver e/ou revisar protótipos.
Passos Alternativos
  1. No passo 1, caso não seja necessário o desenvolvimento de protótipos para validação com cliente, seguir para a atividade implementar solução.
  2. No passo 2, se identidade visual é existente, seguir para o passo 3;
Observações
  • Para projetos de sistemas legados as ferramentas necessárias para o projeto da interface visual serão o Microsoft Visio e/ou a própria Ferramenta de Desenvolvimento de Software da Unidade.
  • Para projetos de sistemas WEB as ferramentas necessárias para o projeto da interface visual serão o Adobe Photoshop, Adobe Flash, Navegadores WEB e/ou a própria Ferramenta de Desenvolvimento de Software da Unidade.
