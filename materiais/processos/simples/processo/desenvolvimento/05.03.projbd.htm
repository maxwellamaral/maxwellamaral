05 - Desenvolvimento

05.03. Projetar ou refinar banco de dados
Finalidade
Analisar entidades e relações entre entidades da base de dados baseados nos requisitos do projeto. Projetar base de dados utilizando os padrões de projeto para construção de diagramas ER conceitual.
Artefatos de Entrada
SMS - Solicitação de Manutenção e Serviços
VIS - Documento de Visão
REQ - Documento de Requisitos
Artefatos de Saída
DER - Diagrama Entidade-Relacionamento (Nível Conceitual)
Papel Responsável:
ANL - Analista de Sistemas
Papel Participante:
Nenhum
Ferramentas
Microsoft Visio
Passos
  1. Fazer análise dos requisitos;
  2. Projetar e/ou refinar banco de dados em um Diagrama Entidade-Relacionamento Conceitual do projeto.
Passos Alternativos
Nenhum
Observações

Esta atividade abordará o projeto da base de dados em um diagrama entidade-relacionamento conceitual. A modificação deste projeto para um diagrama entidade-relacionamento físico acontecerá na atividade implementar solução desta fase do processo.
